# WebDev Year 2 Project

A web-based course management system built with PHP and JavaScript, running in a Docker container.
This is a project is for the second year of the Web Development.

## Features

- **User Management**
    - User registration and login
    - Role-based access (Students and Teachers)
    - User search functionality

- **Course Management**
    - Create and edit courses
    - Add students and teachers to courses
    - Delete courses

- **Course Content**
    - Add text content to courses
    - Upload and manage files
    - Download course files
    - Remove course elements

## Setup

### Prerequisites

- Docker
- Docker Compose

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Run the following command:

```bash
docker-compose up -d
```
or
```bash
docker compose up
```

The application will be available at `http://localhost:80`.

## Configuration

- `php.ini` - PHP configuration settings
- `Dockerfile` - Docker image configuration
- `docker-compose.yml` - Docker services configuration
